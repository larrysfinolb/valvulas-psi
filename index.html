Bienvenid@ a la aplicación web de la Empresa PDVSA.

Esta aplicación permite llevar el historico de las medidas de valvulas de presión (PSI) que se encuentran en diferentes pozos. A continuación se presentaran las diferentes funciones que puedes hacer desde la app.

  1. Función 1: Opción del menu superior con el nombre de "Pozos petroleros". Desde allí podras ver los pozos que se encuentran registrados en sistema y registrar nuevos pozos.
  2. Función 2: Opción del menu superior con el nombre de "Valvulas PSI". Desde allí podras ver las valvulas que se encuentran registradas en sistema y registrar nuevas valvulas.
  3. Función 3: Opción del menu superior con el nombre de "Medidas de las Valvulas PSI". Desde allí podras ver las medidas que se encuentran registradas en sistema y registrar nuevas medidas.
  4. Función 4: Desde "Medidas de las Valvulas PSI" podras filtrar el historico de las medidas PSI de las diferentes valvulas que tiene un pozo.
  5. Función 4: Desde "Medidas de las Valvulas PSI" podras generar un grafico de barras con las medidas pertenecientes a una valvula entre dos fechas.
